## ⚙️ Hardware Components
The smart feeder is engineered with carefully selected components to ensure reliability and cost-effectiveness[cite: 1]:
* **NodeMCU (ESP8266):** IoT microcontroller with Wi-Fi connectivity[cite: 1].
* **Relay Module (5V):** Safely switches the DC motor[cite: 1].
* **DC Motor:** Rotates the feeding drum for dispensing[cite: 1].
* **Buck Converter:** Steps down voltage for component safety[cite: 1].

## 🚀 Key Features
* **Intelligent Automation:** The motor activates for a precise duration of 2–3 seconds to dispense a carefully measured portion[cite: 1].
* **Remote Operation:** Managed via a mobile app featuring a "Feed Now" button and scheduled feed times[cite: 1].
* **Voice Integration:** Integrates seamlessly with Google Assistant (GOOGLE HOME) and Alexa for smart home compatibility[cite: 1].
